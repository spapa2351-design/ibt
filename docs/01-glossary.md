# 도메인 용어

## 한 줄

**모든 문서가 같은 단어를 같은 뜻으로 써야 한다 — 회의에서 30분 새는 90%는 용어 정의 부재에서 온다.**

## 사업·시험 용어

| 용어 | 정의 |
|---|---|
| **KoPASS IBT** | 본 프로젝트의 정식 명칭 (Korean Proficiency Assessment - Internet Based Test). 일본 시장 한정 |
| **TOPIK** | 한국 정부 공인 한국어 능력 시험. KoPASS는 대체가 아닌 보완·진단 위치 |
| **TOPIK 말하기** | TOPIK 내 말하기 영역. 일본 내 시행 불가 구조. **Phase 1의 유일한 타깃** |

## 레벨

| 레벨 | 대상 | 시간 | 문항 | 비고 |
|---|---|---|---|---|
| **초급 (Beginner)** | TOPIK I 수준, 학습 1~2년 미만 | 약 25~30분 | 5영역 / **23문항** / 100점 | 251020 ver.3 기준 |
| **중상급 (Intermediate-Advanced)** | TOPIK II 수준, 학습 2년 이상 | — | — | 영역 구성 미확정 |

## 문제 유형 (초급 5영역, 251020 ver.3 기준)

말하기는 영역별로 콘텐츠 구조·시간·평가축이 다른 **5개 시험 영역**으로 구성. 영역 6은 ver.3에서 삭제됨.

| # | 영역 | 문항 | 배점 | 준비 / 녹음 | 콘텐츠 |
|---|---|---|---|---|---|
| 1 | **짧은 문장 읽기** | 10 | 20 | 0초 / 10초 | 한국어 1문장 (인사·평서·의문·청유·복문) + 음운 규칙 태그 |
| 2 | **질문 대답** | 5 | 30 | 10초 / 20초 | 질문 텍스트 (음성 재생) + 모범 답안 N개 |
| 3 | **생활 속 대화** | 4 | 24 | 20초 / 20초 | 상황 일러스트 + 상대 대사(음성) + 답변 풍선 |
| 4 | **그림/사진 설명** | 3 | 18 | 30초 / 40초 | 이미지 + 긴 모범 답안 |
| 5 | **화용 과제** | 1 | 8 | 40초 / 50초 | 일본어 상황 설명 + 화행(의뢰·사과·감사·조언·요청) |

> 총 23문항 · 100점 · 25~30분. 별도 0번(준비 안내 5분) + 7번(마무리 5분).

영역마다 공통적으로 ① 한국어 지시문 ② 일본어 지시문 ③ TTS(남/여 선택) 또는 mp3 업로드 ④ 준비 시간 ⑤ 비프음('삐') ⑥ 녹음 시간을 포함. 영역 3·4는 이미지 자료가 추가됨. 영역 5는 응시자가 일본어 상황 설명을 읽고 한국어로 표현.

평가축은 영역별로 다름:
- 영역 1: 발음 단일축 (0~2점)
- 영역 2·3·4: 발음 / 어휘·문법 / 내용·구성 3축 (각 0~2점)
- 영역 5: 위 3축 + 화용 능력 4축 (각 0~2점)

## 채점

| 단계 | 주체 | 내용 |
|---|---|---|
| **1차** | AI | 시험 종료 즉시 STT 변환 + 평가 알고리즘 (점수, 1차 피드백 초안) |
| **2차** | Human | 평가자가 AI 결과 검수, 개인화 코멘트 작성 (2인 교차) |
| **확정** | Human | '확정' 버튼 → 성적표 발행 대기 상태 |

평가 차원 5개: **발음 / 유창성 / 어휘 / 문법 / 내용 구성**

> 2차 검수의 운영 시점은 미확정. `open-questions.md` Q6.

## 화면 용어 (응시자)

| 용어 | 정의 |
|---|---|
| **시험 안내** | 응시 시작 전 모달 — 시험명·기본정보·합격기준·시험지 정보·응시기간·문항수 |
| **마이크 테스트** | 응시 전 의무 사전 점검 (마이크 권한·볼륨·5초 녹음 리플레이) |
| **지시어** | 문제별 지시문 텍스트(준비/녹음 시간 안내 포함) |
| **준비 시간** | 답변 구상 시간. 타이머(파랑) + 메모장 활성화 |
| **비프음** | 녹음 시작을 알리는 '삐' 신호음 |
| **녹음(발화) 시간** | 답변 녹음 시간. 타이머(빨강) + "녹음 중" 인디케이터 + Waveform |
| **AI 피드백** | 문항별 STT 결과 + 발음/등급 + 추천 연습 (자음 보강·받침·억양) |
| **재응시** | 시험 결과 화면에서 제출 내역 초기화 후 처음부터 다시 |
| **문제 다시보기** | 시험 결과 화면에서 본인 답안·녹음을 재확인 |

## 화면 용어 (관리자)

| 용어 | 정의 |
|---|---|
| **문제 등록** | 단일 문항 생성/수정. 초급 5영역(짧은 문장 읽기·질문 대답·생활 속 대화·그림 설명·화용 과제) 중 1영역 선택 |
| **문제 은행** | 등록된 문항 풀. 시험지 구성 시 Drag & Drop |
| **시험지 등록** | N문항 단위로 시험지 세트 구성 (배점·순서·미리보기) |
| **시험 관리** | 시험 목록·공개 상태·응시 기간 관리 |
| **응시자 관리** | 시험별 응시자 목록·결과·답안 초기화·결과 공개 |
| **시험 통과 조건** | 최소 응시 문항 수, 개별 시험 점수 기준, 총점 기준, 평균 기준 등 |

## 화면 용어 (평가자)

> ⚠️ 자료엔 글로만 기술됨. 디자인 미존재.

| 용어 | 정의 |
|---|---|
| **3단 분할 뷰** | [문제 정보] · [응시자 답안 (Audio + STT)] · [채점 패널] |
| **STT 싱크** | STT 텍스트 클릭 → 해당 구간 오디오 자동 재생 |

## 기술 용어

| 용어 | 정의 |
|---|---|
| **STT** | Speech-to-Text. AI 채점의 1단계 |
| **TTS** | Text-to-Speech. 지시문 음성 자동 생성 (또는 mp3 직접 업로드) |

## 약속된 표기

- 시간: 24시간 표기 (예: `00:13:24`)
- 점수: `획득점/만점` (예: `30점 / 200점 만점`)
- 등급 표기: 본 결정 전까지 "등급(N단계)" 같이 단계 수 명시

## 혼동하기 쉬운 용어

| ❌ 사용 금지 | ✅ 올바른 용어 | 이유 |
|---|---|---|
| "TOPIK 말하기 시험" (KoPASS 본체를 가리킬 때) | **KoPASS IBT 말하기** 또는 **KoPASS 말하기 진단** | TOPIK은 한국 본사 공인 시험. KoPASS는 별도 진단형. 혼용 시 공인 여부 오인 |
| "한국어 말하기 시험" | **KoPASS IBT 말하기** | 일반명사여서 검색·문서 트래킹 어려움 |
| "모의고사" (콘텐츠가 아닌 시스템 명칭으로 쓸 때) | **진단평가** 또는 **KoPASS** | 모의고사는 콘텐츠 종류. 플랫폼·시스템 명칭으로는 모호 |
| "유형 1·2형" / "TOPIK 말하기 1형/2형" (UI 라벨) | **영역 1~5** (영역명 사용 — 짧은 문장 읽기, 질문 대답 등) | 251020 ver.3 기준 시험 구조는 "유형"이 아닌 "영역" 단위 |
| "어플" / "어플리케이션" | **앱** | 일본어 표기와 정합. 안내 문구 톤 일관성 |
| "폰" / "스마트폰 앱" | **iOS/Android 앱** | OS 명시. "스마트폰"은 모바일 웹과 혼동 |
| "오픈" (서비스 개시 의미) | **서비스 개시** 또는 **런칭** | "오픈"은 모호 — 회원가입 오픈/시험 오픈 등 다른 의미와 충돌 |
| "회차" (시행 회차 의미) | **시행 회차** 또는 **회차** (일관) | 응시자가 푸는 *문항 번호*와 혼동 가능 — 문항 번호는 *문항 N번*으로 |
| "유저" / "사용자" (관리자 vs 응시자 구분 없을 때) | **응시자** / **관리자** / **평가자** 명시 | 권한·동선이 다르므로 추상화 금지 |
| "테스트" (시험 의미로) | **시험** 또는 **응시** | "테스트"는 QA·환경 점검과 혼동 |
| "녹화" | **녹음** | 음성만 수집. 영상 녹화 기능 없음 |
