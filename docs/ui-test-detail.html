📚 시험 소개

시험은 5개 영역(발음·유창성·어휘·문법·내용 구성)으로 구성되며, 총 23문항을 약 25분 안에 응시합니다. 모든 답변은 자동 녹음되어 AI 채점 후 즉시 결과가 제공됩니다.

합격 기준은 총점 기반으로 1급~6급 등급이 부여되며, 영역별 5각 차트로 약점이 시각적으로 표시됩니다.
