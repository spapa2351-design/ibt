전체 자주 묻는 질문

총 14개 · 가장 많이 검색된 질문 순으로 정렬됩니다.

결제·환불 결제 후 응시하지 못한 경우 환불받을 수 있나요? +

접수 기간 내 미응시인 경우 마이페이지 → 1:1 문의에서 환불 신청이 가능합니다. 운영팀 검토 후 영업일 기준 3~5일 이내 결제 수단으로 환원됩니다.

응시 가능 기간(14일) 만료 또는 응시 후 변심의 경우는 환불이 적용되지 않습니다.

단, 시스템·네트워크 장애로 응시가 정상 진행되지 못한 경우는 자동 환불 또는 재응시 티켓이 발행됩니다.

응시 환경 응시 가능한 디바이스는 무엇인가요? +

데스크탑 브라우저 (Chrome, Safari, Edge — 최신 버전 권장) 또는 iOS/Android 앱에서 응시 가능합니다.

마이크 권한이 필수이며, 헤드셋 사용을 권장드려요. 모바일 브라우저는 폴백으로 동작합니다.

응시 환경 시험 도중 마이크가 작동하지 않으면 어떻게 하나요? +

응시는 자동으로 계속 진행되며, 답변은 단말 측에 자동 저장됩니다. 시험 종료 후 운영팀에 문의하시면 로그 확인 후 재응시 티켓이 발행됩니다.

결제·환불 결제는 어떤 수단으로 가능한가요? +

クレジットカード(VISA·JCB·MasterCard)와 PayPay 결제를 지원합니다. 결제 시 영수증 PDF가 자동 발행되어 마이페이지 → 결제 내역에서 다운로드 가능합니다. (적격청구서 인보이스 PDF는 추후 PG API 연동 시점에 도입 예정)

인증서 인증서는 어떻게 발급받나요? +

응시 후 결과 발표가 되면 마이페이지 → 인증서에서 PDF를 다운로드할 수 있어요. 위변조 검증 URL이 함께 발급되어 외부에서도 진위 확인이 가능합니다.

응시 환경 응시 직전 환경 점검은 왜 필요한가요? +

본 시험에서 마이크·네트워크 문제로 응시가 어려워지는 사고를 방지하기 위해, 마이크 권한·음량·5초 녹음 재생·네트워크 점검을 시험 시작 직전 게이트로 확인합니다. 통과해야 응시 클라이언트에 진입할 수 있어요. 결제 시점과는 분리되어 있습니다.

시험 콘텐츠 초급과 중상급의 차이는 무엇인가요? +

초급은 TOPIK I 수준(학습 1~2년 미만)으로 약 25분 23문항, 중상급은 TOPIK II 수준(2년 이상)으로 약 35분 14문항입니다. 의견 제시·롤플레이·프레젠테이션 등이 중상급에 포함됩니다.
