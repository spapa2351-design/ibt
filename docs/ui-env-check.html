마이크가 잘 인식되는지 확인합니다

"안녕하세요, KoPASS 응시 환경 점검입니다" 라고 말해 보세요. 아래 게이지가 함께 움직이면 정상입니다.

● 녹음 중 — 말소리가 잘 잡힙니다
현재 입력 음량 -18 dB · 권장: -24dB ~ -6dB
⚠ 마이크 권한이 거부되었습니다
브라우저 주소창의 마이크 아이콘을 눌러 권한을 허용해 주세요. (Chrome / Safari / Edge 모두 동일) 권한 허용 후 "다시 시도"를 눌러 주세요.

💡 정확한 점검을 위한 팁

①헤드셋 권장 — 노트북 내장 마이크보다 인식률이 높습니다.
②조용한 환경 — TV·라디오·환풍기 꺼주세요.
③입과 마이크 거리 10~15cm 유지.
④다른 통화·녹음 앱 종료 — 마이크를 점유하면 인식 실패합니다.
마이크 권한을 거부했나요? 주소창의 🔒 자물쇠 아이콘 → 사이트 설정 → 마이크 → 허용 로 변경하세요.
