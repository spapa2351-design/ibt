FAQ

자주 묻는 질문

응시 가능한 디바이스는 무엇인가요?+
1차 오픈은 데스크탑 브라우저 (Chrome, Safari, Edge · 최신 버전 권장) 와 모바일 브라우저에서 응시 가능합니다. 마이크 권한이 필수이며, 헤드셋 사용을 권장드려요. iOS / Android 네이티브 앱은 라운드 2 이후에 제공될 예정입니다.
결제 후 언제까지 응시할 수 있나요?+

결제일로부터 14일 이내 24시간 언제든 응시 가능합니다. 시험 시작 시각이 별도로 정해져 있지 않아 본인의 일정에 맞춰 진행할 수 있어요.

응시 가능 기간이 지나면 자동 만료되며 환불되지 않으니 기간 내에 응시해 주세요.

환불 정책은 어떻게 되나요?+

접수 기간 내 미응시인 경우 마이페이지 → 1:1 문의에서 환불 신청이 가능합니다. 운영팀 검토 후 영업일 기준 3~5일 이내 결제 수단으로 환원됩니다.

응시 후 변심 또는 결과 불만족의 경우는 환불이 적용되지 않습니다. 단, 시스템·네트워크 장애로 응시 진행 불가 시는 자동 환불 또는 재응시 티켓이 발행됩니다.

인증서는 어떻게 발급받나요?+
응시 후 결과 발표가 되면 MaroPass 마이페이지 → 인증서에서 PDF 인증서를 다운로드할 수 있어요. 위변조 검증 URL이 함께 발급되며, 검증 페이지는 kopass.* 도메인이라 외부 학교·기업이 직접 MaroPass에 접속해 진위를 확인할 수 있습니다.
시험 도중 마이크가 작동하지 않으면?+

응시는 자동으로 계속 진행되며, 답변은 단말 측에 자동 저장됩니다. 시험 종료 후 운영팀에 문의하시면 로그 확인 후 재응시 티켓이 발행됩니다.

응시 시작 직전 환경 점검 게이트(마이크·네트워크·브라우저)를 통과해야 응시 클라이언트에 진입하므로, 시험 시작 버튼을 누른 후 표시되는 점검을 꼭 통과해 주세요. 한 번 통과한 환경 점검은 동일 기기에서 30일간 유효해서, 같은 기기로 다시 응시할 때는 점검이 생략됩니다.

응시는 어디에서 진행되나요?+

마이페이지 → 나의 시험에서 응시하기를 누르면 응시 전용 화면이 새 창에서 열려요. 결제 영수증·인증서 PDF 같은 보관 자료는 그대로 MaroPass 마이페이지에서 다운로드합니다.

새 창이 열리지 않으면 브라우저의 팝업 차단 허용이 필요해요. 응시 중에도 MaroPass는 그대로 떠 있어서 결제 내역 등을 같이 확인할 수 있습니다.
