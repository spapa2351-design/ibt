1. 개요

본 문서는 MaroPass IBT(일본인 한국어 학습자 대상 '말하기' 진단평가) 정식 오픈까지의 작업을 분해하여 담당·선행관계·공수를 산정한 작업분해구조서임. MaroPass는 TOPIK을 대체하는 시험이 아니라 학습자의 말하기 능력을 진단·보완하는 IBT 상품이며, 운영은 뉴인재팬(Newin Japan), 서비스 브랜드는 MaroPass(kopass.*)로 함.

WBS 기준선은 Phase 1(2026년 12월 초 정식 오픈, 11월 초 테스트 납품) 범위로 한정함. 앱(iOS/Android), 2차 인간 평가자 UI, 푸시 알림 등은 라운드2 항목으로 본 기준선에서 제외하며, 별도 표기함. 산정 단위는 man-day(M/D)이고, 1 M/D = 1인 기준 실작업 1일(회의·리뷰 제외)로 봄. 추정치는 확정 스펙 기준이며, 미확정 항목(결제 PG, 등급체계)은 확정 후 재산정 전제임.

팀 구성 전제: PM 1(박지원), 디자인 2(이수민 외 1), 프론트 2, 백엔드 2, QA 1(최유진), 운영 1. 개발 리드(김도현)는 백엔드 겸임. 외부는 일본어 네이티브 검수(외주), 한국교육재단(등급 협의)임. 기존 터치클래스 사용자/관리자 web 자산을 최대한 재사용하고, MaroPass 고유 영역만 신규 개발하는 것을 원칙으로 함.

6. 가정 및 제약

  • 기존 터치클래스 사용자/관리자 web 자산 재사용 전제. MaroPass 고유 영역(응시·신규 메뉴 5개·5영역 탭)만 신규 개발.
  • 관리자 시험관리·문제은행·응시자관리·채점·통계·설정은 기존 그대로 사용(개발 대상 아님).
  • Phase 1은 웹 only(데스크탑 + 모바일 웹). 네이티브 앱은 라운드2.
  • 2차 인간 평가자 UI, 푸시 알림은 라운드2/Premium으로 본 기준선 제외.
  • 결제 후 응시기간 14일, 24시간 상시 응시 가능. 무료 샘플 3문항 제공.
  • 채점은 AI 1차 자동 → 운영자 '결과 공개' 수동 트리거 방식. 자동 공개 아님.
  • man-day는 실작업 기준(회의·리뷰·대기 제외). 완료 표기 항목(3.2/3.3/3.4 등)은 시안 산출물 기준 완료이며, 개발 반영은 별개.
  • 추정 합계(442 M/D)는 5.0 PG 미확정 잠정치. PG·등급 확정 시 재산정.
