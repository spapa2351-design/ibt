# MaroPass IBT 중간보고 v2 — 서비스 사이트

> KOEN(코엔, 一般社団法人) 측 공유용 **중간보고**. v1(2026-05 전달)의 후속 버전으로, 이번에는 **서비스 사이트**의 실제 화면을 공유합니다. Newin 내부 SSOT와는 별개의 *외부 보고용* 문서입니다.

## 이번 보고의 초점

v1은 시스템 구조·응시자 플로우·운영 정책·역할 분담을 합의하고 화면은 **박스 와이어프레임**이었습니다.
v2는 그중 **서비스 사이트(MaroPass 본 사이트)** — 가입·결제·마이페이지 등 학습자 회원·결제 영역 — 의 **실제 화면 15종**을 링크 카드로 공유합니다.
응시(사용자 web)·운영(관리자 web) 화면은 다음 보고에서 이어집니다.

## 패키지 구성

| 파일 | 용도 |
|---|---|
| `index.html` | 중간보고서 — 이번 보고 내용·진행 현황·서비스 사이트 화면(15) 링크 카드·플로우 상 위치·다음 단계 (한/일 토글, 인쇄/PDF 지원) |
| `flow-overview.html` | 보조 — 서비스 사이트 화면 흐름도(구조 뷰). 구매 여정 + 마이페이지 허브를 박스 다이어그램으로, 각 박스는 실제 화면으로 여는 링크 |

## 서비스 사이트 화면 (15)

- **진입·회원 (3)** — 메인 홈, 회원가입·로그인, FAQ
- **시험·결제 (4)** — 시험 목록, 시험 상세, 결제, 결제 완료
- **마이페이지 (8)** — 마이페이지 홈, 응시할 시험, 결과, 인증서, 결제 내역, 환불, 1:1 문의, 프로필

각 카드의 **화면 열기 ↗** 버튼은 `../../ui/service-site/`의 실제 화면을 새 탭으로 엽니다.

## ⚠️ 전달(패키징) 시 유의

링크 카드는 `../../ui/service-site/` 실제 화면 파일을 참조합니다. KOEN에 전달할 때는 **`ui/` 폴더(service-site·shared·user-web 포함)를 이 보고서와 같은 상대 구조로 함께 전달**해야 화면 열기가 동작합니다. (일부 화면이 `../shared/`, `../user-web/`를 상호 참조)

## 사용 안내

- 화면의 문구·데이터·금액은 **예시 값**이며, 색상·레이아웃 등 세부 디자인은 검수 과정에서 조정될 수 있습니다.
- 메인 홈은 최신 시안(v0.2)을 연결했습니다.
